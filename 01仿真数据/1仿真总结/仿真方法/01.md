## 正交实验设计方法

### 行李箱操作人机工程研究的多因素挑战

行李箱作为日常出行与长途旅行的必备工具，其操作的便捷性与舒适性直接关系到用户体验。然而，行李箱的人机交互过程是一个复杂的多因素系统，其性能表现受到使用者个体特征、操作环境、行李自身属性以及具体操作方式等多维度变量的综合影响。

1.  **影响因素众多且层级复杂**：初步分析与文献回顾表明，潜在影响因素不仅数量庞大，且涵盖多个层级。从**人体因素**（如性别、年龄段、身高、体重、腰椎曲度、核心肌群力量、上肢关节活动度）、**环境因素**（如路面平整度与材质、环境坡度、是否存在障碍物、空间拥挤程度）、**操作行为因素**（如推/拉姿态、提/放姿态、转向操控方式、行进速度、加速度变化、持续操作时间、是否伴随振动）到**行李设备因素**（如行李总质量、重心分布、箱体尺寸、拉杆长度与角度、轮径大小与材质、轮子布局）等，均可能对使用者的生理负荷与主观感知产生显著影响。

2.  **全因素实验设计的"组合爆炸"与实践困境**：若试图对上述所有潜在因素及其不同水平进行全面的全因素实验设计，实验组合的数量将呈现指数级增长，迅速达到一个在实际操作中无法承受的量级。即便仅选取本研究后续聚焦的5个核心因素（使用者性别2水平、人体百分位5水平、路面类型3水平、操作姿态5水平、行李质量3水平），其全排列组合已达 $2 \times 5 \times 3 \times 5 \times 3 = 450$ 种工况。若要对每种工况进行充分的生物力学仿真或真人实验，所需的时间成本、计算资源以及数据分析的复杂度将是巨大的，这使得传统全因素方法在解决此类多变量优化问题时显得力不从心。

3.  **因素间交互效应的普遍性与复杂性**：更为复杂的是，各因素之间并非独立作用，而是普遍存在交互效应。例如，高百分位的使用者在粗糙路面上操作重型行李箱时所承受的肩部力矩，并不能简单地通过各单因素影响的线性叠加来预测。这些交互作用使得准确评估和优化整体系统性能变得更具挑战性。

面对上述挑战，寻求一种能够在保证研究科学性与结论可靠性的前提下，显著减少实验规模、提高研究效率的实验设计方法，成为本研究的首要任务。正交实验设计（Orthogonal Experimental Design），特别是基于田口方法（Taguchi Methods）的稳健参数设计，为此类多因素、多水平系统的研究提供了一种高效且实用的解决方案。

### 因子选择与水平划分的理论依据

为了确保正交实验设计的针对性与有效性，本研究在因子选择与水平划分上进行了审慎的考量。结合对 `行李箱场景参数.csv`、`人体测量数据.csv` 及 `关节数据.csv` 等基础数据库的深度挖掘，并参照人因工程学相关文献与行业标准，最终筛选出以下五个对行李箱操作负荷具有显著影响且易于控制和量化的关键因子：

1.  **使用者性别 (Factor $F_1$, 2 Levels: 男性, 女性)**
    *   **理论依据**: 性别是导致个体间生理特征差异的首要因素。男女在骨骼结构、肌肉质量与分布、脂肪含量、激素水平等方面存在显著差异，这些差异直接体现在身高、体重、肢体长度、关节活动范围以及绝对力量和耐力上。例如，相关人体测量数据库显示，成年男性的平均上肢力量通常比女性高出30%-60%。因此，将性别作为独立因子进行考察，对于准确评估不同用户群体的生物力学响应至关重要。
    *   **水平设定**: 选用"男性"与"女性"两个离散水平，代表了最基本且广泛的人群分类，确保了研究结果对两大主要用户群体的普适性。

2.  **人体百分位 (Factor $F_2$, 5 Levels: P5, P10, P50, P90, P95)**
    *   **理论依据**: 人体百分位是人因工程设计中用以表征人群身体尺寸多样性的核心概念。它描述了在特定人群中，某个身体尺寸值小于或等于该百分位数值的个体所占的百分比。仅考虑平均值（如P50）无法覆盖人群的广泛差异，尤其在与体力相关的任务中，极端体型（如P5或P95）的用户往往面临更大的挑战。本研究旨在通过选取多个关键百分位点，来模拟不同体型用户在操作行李箱时的潜在差异。
    *   **水平设定**: 选择P5、P10、P50、P90和P95五个水平，是基于以下考虑：
        *   **覆盖广度**: P5和P95代表了人群中体型较小和较大的近极端群体，确保了设计对绝大多数用户的适应性。
        *   **代表性**: P50代表中位数或平均体型用户。
        *   **分布细节**: P10和P90的加入，进一步细化了对人群分布的模拟，能够更全面地捕捉体型变化对操作负荷的影响趋势。例如，从查阅的`人体测量数据.csv`可见，P5女性与P95男性在关键的上肢肢段长度（如上臂长、前臂长）和整体质量上，差异可达25%-40%，这种差异将直接影响到杠杆力臂和惯性参数，进而影响关节力矩。

3.  **路面类型 (Factor $F_3$, 3 Levels: 机场平面, 斜坡, 粗糙地面)**
    *   **理论依据**: 路面是行李箱操作的直接承载界面，其物理特性（如摩擦系数、平整度、坡度）直接决定了克服滚动阻力、坡度阻力以及应对颠簸冲击所需的手部作用力。不同路面类型对行李箱的移动平顺性和稳定性有显著影响，进而影响用户的操控感受和生理负荷。
    *   **水平设定**: 基于对实际使用场景的调研（参考`行李箱场景参数.csv`中不同场景的参数设定）和文献分析，确定了三类典型且具有代表性的路面类型：
        *   **机场平面**: 特点是表面平整、硬度高、滚动摩擦系数小（如 $\mu_{roll} \approx 0.015 - 0.03$）。这代表了理想或常见的室内使用环境，如机场大厅、火车站台等。
        *   **斜坡**: 设定为约8°的倾角。这一角度参考了公共场所无障碍设计规范中对坡道坡度的常见限制，代表了用户在进出站口、连接通道等场景下可能遇到的情况。
        *   **粗糙地面**: 特点是表面不平整、可能存在障碍物、滚动摩擦系数较大（如 $\mu_{roll} \ge 0.05$），并且操作时会伴随额外的振动和冲击效应。这代表了户外人行道、石板路或一些维护不佳的路面。
        这三种水平基本覆盖了行李箱在不同环境下的主要受力特征变化。

4.  **操作姿态 (Factor $F_4$, 5 Levels: 水平推进, 斜坡拉行, 侧向拉提, 转向操作, 越障动作)**
    *   **理论依据**: 用户在操作行李箱时会根据具体任务和个人习惯采用不同的身体姿态，而不同的姿态直接决定了上肢各关节的角度配置，从而改变了外力作用于人体的力臂和肌肉的募集模式。例如，推拉行李时肩、肘、腕关节的协同模式与提箱上下台阶时的模式截然不同。
    *   **水平设定**: 依据对用户日常操作行为的观察、生物力学文献中对上肢作业姿态的分类以及`关节数据.csv`中记录的典型操作关节角度，定义了五种具有代表性的操作姿态：
        *   **水平推进**: 例如在平地上双手或单手从后方推行行李箱。
        *   **斜坡拉行**: 例如在上坡或平地上单手从前方或侧前方拉行行李箱。
        *   **侧向拉提**: 例如将行李箱从地面提起或进行短距离侧向搬运。
        *   **转向操作**: 例如在行进中改变方向时，身体和上肢产生的扭转与推拉组合动作。
        *   **越障动作**: 例如提拉行李箱上下台阶或通过路沿。
        这些姿态旨在模拟用户在不同场景下的主要操作方式，覆盖了从稳态行走到动态调整的多种情况。

5.  **行李质量 (Factor $F_5$, 3 Levels: 8kg, 14kg, 20kg)**
    *   **理论依据**: 行李质量是产生外部负荷最直接的来源。质量越大，克服其惯性、重力分量和摩擦力所需的作用力就越大，对人体关节和肌肉的负荷也相应增加。行李质量的选择需要考虑航空公司的行李规定、用户的携带习惯以及不同出行目的下的典型载重量。
    *   **水平设定**: 基于对航空公司免费托运及手提行李限额的研究，以及用户调研中关于行李重量的反馈，设定了三个具有实际意义的质量水平：
        *   **8kg**: 代表常见的手提行李或短途轻载情况。
        *   **14kg**: 代表中等重量的行李，常见于一般差旅或短期度假。
        *   **20kg**: 接近或达到多数经济舱的免费托运上限，代表重载情况。
        这三个水平覆盖了从轻载到重载的常见范围，有助于评估不同负荷等级下的人体响应。

### L50正交表的选择理由与实施策略：为何选择L50及自定义脚本的价值

#### L50(2^1 × 5^11)正交表的选择考量

在众多可选的正交表类型中，本研究最终选定L50(2^1 × 5^11)标准正交表，主要基于以下综合考量：

1.  **与研究因子结构的高度匹配性**: 本研究的核心因子包含一个2水平因子（使用者性别）和四个多水平因子（人体百分位5水平、路面类型3水平、操作姿态5水平、行李质量3水平）。L50表恰好提供了一个2水平的列和多达11个5水平的列，这使得我们可以直接将性别因子对应到2水平列，并将其他四个因子灵活地映射到5水平列上。这种结构上的天然契合简化了实验设计过程，避免了复杂的因子拆分或虚拟水平处理。

2.  **实验规模的显著优化与代表性保证**: 如前所述，全因素实验设计将产生450种组合。L50正交表通过其精巧的数学构造，能够以50组精心挑选的实验组合来代表这450种可能性。这意味着实验规模缩减了将近90% ($ (450-50)/450 \approx 88.9\% $)，极大地节省了后续生物力学仿真所需的计算资源和时间成本。同时，正交表的特性确保了这50组实验在因子水平的分布上具有良好的均衡性和代表性，能够有效地覆盖主要的设计空间。

3.  **主效应分析的清晰性与统计效能**: L50正交表的设计保证了各列之间的正交性（或近似正交性），这意味着不同因子（或其不同水平）对输出结果（如关节力矩）的影响可以被相对独立地进行评估。这种特性极大地简化了后续的数据分析过程，使得研究者可以更清晰地识别出哪些因素是影响操作负荷的关键因素，以及这些因素的最佳水平组合，为产品优化提供直接依据。

4.  **处理混合水平因子的灵活性**: 尽管人体百分位和操作姿态因子本身是5水平，但路面类型和行李质量因子实际上是3水平。L50表的5水平列设计为这种混合水平映射提供了便利。通过合理的映射策略（详见下一节），可以将实际的3个物理水平非均衡但有意义地分配到L50表的5个逻辑水平上，既满足了正交表对水平数的要求，也反映了实际场景中不同水平出现的频次或重要性差异。

#### 自定义Python脚本进行实验设计与仿真的必要性与优势

尽管市面上有成熟的统计分析软件（如Minitab, SPSS）可以辅助进行正交实验设计，并且存在专业的生物力学仿真平台（如AnyBody, OpenSim），本研究依然选择通过自主编写Python脚本来完成从实验设计到核心生物力学计算的全流程。这一决策基于以下考量，旨在克服通用软件在特定研究情境下的局限性，并最大化研究的精度与深度：

1.  **精确控制与高度定制化的实验设计参数**: 自定义脚本允许研究者对实验设计的每一个细节进行完全的编程控制。例如，在将实际的3水平因子（路面、质量）映射到L50表的5个逻辑水平时，我们可以根据先验知识或特定研究目的，灵活设定非均衡的映射比例（如本研究采用的2:2:1），而不是受限于通用软件可能提供的均等分配或固定模式。这种控制力对于确保实验设计最大程度地贴合研究假设和现实情境至关重要。

2.  **模拟特定且复杂的真实世界交互效应**: 行李箱操作中的许多细微但重要的物理现象，如不同路面引起的微振动、用户在特定姿态下为维持平衡而产生的身体代偿、以及长时间操作可能导致的肌肉疲劳累积效应，很难在标准化的生物力学软件中得到精确建模。通过自定义脚本，研究者可以依据最新的文献研究成果或专门的物理模型，将这些复杂因素的数学表达直接嵌入到仿真计算中（例如，在外部载荷计算中引入冲击因子 $k_{impact}$ 和振动因子 $k_{vibration}$，在疲劳评估中构建特定的 $P_{fatigue}$ 模型）。这种针对性的建模能力是通用软件难以比拟的。

3.  **与自研生物力学模型的无缝集成与迭代优化**: 本研究的核心之一是开发一套专门针对行李箱操作场景的生物力学计算模型。自定义实验设计脚本可以直接生成与该模型输入要求完全匹配的参数组合，实现数据流的无缝对接。更重要的是，在研究过程中，如果生物力学模型需要调整或增加新的参数（例如，更精细的拉杆角度变量、考虑地面不平度的随机扰动输入），实验设计脚本可以同步、快速地进行修改和扩展，支持敏捷的科研迭代。这种紧密的耦合和灵活性是采用分离的第三方软件所难以实现的。

4.  **研究过程的透明度、可复现性与知识积累**: Python作为一门开源且广泛应用的编程语言，其代码本身即可作为研究方法和计算过程的精确记录。这不仅极大地提高了研究的透明度和可复现性（其他研究者可以阅读和运行代码来验证结果），而且将整个研究过程中的模型假设、参数选择、计算逻辑都固化为可管理、可传承的知识资产。这对于后续研究的深入和拓展具有长远价值。

5.  **避免"黑箱"操作，深化机理理解**: 商业软件包往往将复杂的算法封装在内部，用户可能仅知其然而不知其所以然。通过从底层构建计算模型，研究者被迫深入思考每一个环节的物理意义和数学实现，这有助于更深刻地理解行李箱操作背后的人机交互机理，而不仅仅是获得一个仿真结果。

综上所述，虽然自定义脚本开发需要投入额外的时间和精力，但其在精确控制、模拟复杂度、集成灵活性、过程透明度以及深化机理理解方面带来的益处，对于实现本研究的特定科学目标而言是不可或缺的。

### 实验方案的生成过程

基于L50正交表和定义的映射关系，本研究采用以下算法生成实验方案：

```python
def generate_l50_experimental_design(l50_data):
    """基于L50正交表和因子映射关系生成50组实验设计方案"""
    experiments = []
    for i, row_levels in enumerate(l50_data):
        exp_id = f"E{i+1:03d}"  # 格式化为E001, E002, ... E050

        gender_level_oa = row_levels[0]        # X1: 性别
        percentile_level_oa = row_levels[1]    # X2: 百分位
        surface_level_oa = row_levels[2]       # X3: 路面类型
        posture_level_oa = row_levels[3]       # X4: 操作姿态
        weight_level_oa = row_levels[4]        # X5: 行李重量

        # 创建实验设置字典
        experiment_details = {
            "实验编号": exp_id,
            "性别": gender_map[gender_level_oa],
            "百分位": percentile_map[percentile_level_oa],
            "路面": surface_map[surface_level_oa],
            "姿态": posture_map[posture_level_oa],
            "重量": weight_map[weight_level_oa]
        }
        experiments.append(experiment_details)

    return pd.DataFrame(experiments)
```

这一实现确保了每组实验都被唯一编号(E001-E050)，并包含完整的五因子水平组合。生成的实验方案通过以下步骤保存为CSV文件，作为后续生物力学模拟的输入：

### 正交设计的理论优势与实际收益

本研究采用的L50正交设计相比传统方法具有以下优势：

#### 理论优势
1. **信息利用率高**：使用50组组合即获得了对450种可能情况的有效表征，样本利用率提高了约9倍。
2. **因子效应分离**：通过正交设计，每个因子的效应可以相对独立地评估，减少了混杂效应。
3. **数据效率最大化**：在给定计算资源限制下，获取最具代表性的实验数据集。

#### 实际收益
1. **计算资源节约**：仅需计算50组生物力学模拟，而非450组，大幅减少了计算时间和资源消耗。
2. **结果解释简化**：正交设计产生的数据更易于分析各因子的主效应，简化了后续的统计分析过程。
3. **边界条件覆盖**：尽管组合减少，但仍然覆盖了所有关键因子的极值组合，确保了设计的稳健性。
